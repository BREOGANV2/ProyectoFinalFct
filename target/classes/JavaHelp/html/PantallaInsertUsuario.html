Insertar Usuario

En esta pantalla puedes registrar un nuevo usuario en el sistema. Para ello, es necesario completar todos los campos del formulario.

  • Nombre: Nombre completo del usuario.
  • Contraseña: Contraseña para el usuario.
  • Edad: Edad en años (solo números).
  • Género: Selección entre las opciones disponibles (por ejemplo, Masculino, Femenino, Otro).

Una vez completados todos los datos, pulsa el botón "Guardar" en la interfaz de la aplicación para registrar el nuevo usuario (Guardara tambien la fecha del registro).
