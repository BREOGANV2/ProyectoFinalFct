Modificar Ejecución de Rutina

Esta pantalla permite modificar los datos de una ejecución previamente registrada de una rutina.

1. Seleccionar una ejecución

En la parte superior de la ventana encontrarás una tabla con todas las ejecuciones almacenadas. Haz clic sobre una fila para cargar los datos de esa ejecución en los campos editables:

  • Duración (en minutos)
  • Fecha (en formato yyyy-MM-dd)
  • Notas (comentarios u observaciones)

2. Editar los datos

Puedes modificar cualquiera de los campos cargados. El sistema detectará automáticamente si hubo cambios en los valores originales.

3. Guardar los cambios

Una vez realizados los cambios, pulsa el botón "Modificar". Si se detectaron cambios válidos, se actualizará el registro correspondiente en la base de datos y la tabla se refrescará con la información actualizada.

Si no se ha seleccionado una ejecución o no se han hecho cambios, se mostrará un mensaje informativo.
