Insertar Ejercicio

En esta pantalla puedes agregar un nuevo ejercicio a la base de datos. Es obligatorio completar todos los campos del formulario antes de guardar.

  • Nombre: Nombre del ejercicio (por ejemplo: Sentadilla, Press de banca).
  • Grupo muscular: Parte del cuerpo que trabaja el ejercicio (Pecho, Espalda, Piernas, etc.).
  • Descripción: Breve explicación sobre cómo realizar el ejercicio.
  • URL de imagen: Selección de una imagen representativa del ejercicio mediante un botón que abre un explorador de archivos (FileChooser).

Una vez completado el formulario, pulsa el botón "Guardar" disponible en la interfaz de la aplicación para registrar el ejercicio.
