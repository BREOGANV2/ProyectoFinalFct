Pantalla Principal

La pantalla principal de GymTracker se divide en cuatro paneles, cada uno representando una tabla principal de la base de datos del sistema:

  • Usuarios: Muestra la información de todos los usuarios registrados en el sistema (nombre, edad, etc.).
  • Ejercicios: Contiene la lista de ejercicios disponibles, incluyendo su nombre, tipo, grupo muscular y detalles técnicos.
  • Rutinas: Gestiona las rutinas predefinidas o personalizadas, compuestas por varios ejercicios agrupados.
  • Ejecuciones: Muestra el historial de ejecución de rutinas por parte de los usuarios, con fecha, duración, y resultados.

Dentro de cada uno de estos paneles, encontrarás un botón que permite acceder a una pantalla específica con opciones para:

  • Ver: Ver los registros existentes.
  • Añadir: Añadir nuevos datos a la tabla correspondiente.
  • Modificar: Modificar los datos ya existentes.
  • Eliminar: Eliminar registros innecesarios o erróneos.

Esta estructura permite una navegación intuitiva y una gestión clara de toda la información dentro del sistema.
